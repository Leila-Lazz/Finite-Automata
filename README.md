# Finite Automata Project (C)

## Overview
A C program that manipulates **finite automata** loaded from text files. It supports displaying automata, standardization, determinization + completion, minimization, complement automaton generation, and word recognition (acceptance testing) through a menu-driven CLI. [file:72]

## Features
- Read automata from text files
- Standardize automata when needed
- Determinize and complete automata
- Minimize automata
- Build the complementary automaton
- Recognize words (returns accepted/rejected)

## Tech stack
- C language
- Standard C library
- (Optional) GCC / Clang

## Project structure
- `Library.h`: structs + function prototypes
- `Library.c`: implementation of automata functions
- `main.c`: CLI / user interaction
- `automata/` or input files: text files describing automata (add your folder name)
